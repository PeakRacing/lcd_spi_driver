## 介绍
​	适用于spi驱动的lcd通用驱动，考虑分辨率与内存原因，仅支持RGB565

**支持驱动:**

- [x] ST7789V2

- [x] ST7735S

- [x] ST7796S

- [x] ILI9341

- [x] ILI9486

- [x] GC9A01

- [x] NV3030B
